Welcome to python-ovs-vsctl's documentation!
============================================

``python-ovs-vsctl`` provides objects for OVSDB requests by wrapping
``ovs-vsctl`` command which included in `Open vSwitch`_.

Internally, ``python-ovs-vsctl`` does this by just invoking ``ovs-vsctl``
command.

.. _Open vSwitch: http://openvswitch.org/
